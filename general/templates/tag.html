Tags Application

This section describes the API endpoints for managing tags.

Base URL: http://127.0.0.1:8000/

Get Posts By Tag Name

  • Method: GET
  • Endpoint: /tags/posts/<tag-name>/
  • Path Variable:
    • <tag-name>: The name of the tag to retrieve information about.
  • Response:
{
  "posts": [
    # List of posts in tag name
  ]
}
  • Description: Get a list of posts in tag name.
